# Designer Briefs

Each file in this folder is a self-contained brief for one journey. Paste the contents directly into Claude or Lovable when starting your prototype.

The brief includes everything you need: your story beats, specific data, conversation examples, shared constraints, and what to avoid. No need to cross-reference other documents.
