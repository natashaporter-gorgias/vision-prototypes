# Gorgias AI Agent — Design Prototypes

Interactive prototypes for the Gorgias AI Agent vision, built by the Product Design team.

## Prototypes

| Prototype | Description |
|-----------|-------------|
| [Vision v2](index.html) | AI Agent dashboard with 4 customer journeys (UrbanStems) |
| [Vision v2 — Preview](preview-v2.html) | Standalone preview of the v2 prototype |
| [Vision v3 — Conversational](gorgias-vision-v3.jsx) | Conversational evaluation flow, inspired by Sierra's "Agents as a Service" |
| [Evaluations Flow](eval-flow-prototype.html) | AI Agent evaluation and testing flow |

## Viewing locally

Open any `.html` file directly in your browser — no build step required. These are self-contained prototypes using React via CDN + Babel standalone.

## GitHub Pages

This repo is set up for GitHub Pages. Once enabled, prototypes are viewable at:

```
https://<your-username>.github.io/gorgias-prototypes/
```

## Tech stack

- HTML / CSS / JS (vanilla, no build step)
- React 18 (via CDN)
- Babel standalone (for JSX transpilation in-browser)
- Gorgias Axiom design tokens
